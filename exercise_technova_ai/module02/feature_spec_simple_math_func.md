# 功能规约：计算矩形面积

**目标**: 编写一个 Python 函数 `calculate_rectangle_area`。

**输入**:
*   `length`: 矩形的长度 (预期为数字)。
*   `width`: 矩形的宽度 (预期为数字)。

**输出**:
*   返回计算出的面积 (长度 * 宽度)。

**行为要求**:
*   如果 `length` 或 `width` 不是正数（小于或等于 0），函数应返回 0。
*   (可选加分项) 如果输入非正数，函数还应使用 `warnings.warn()` 打印一条警告信息，提示输入无效。
*   函数必须包含符合 PEP 257 风格的 Docstring，清晰说明其功能、参数 (`Args:`) 和返回值 (`Returns:`)。
